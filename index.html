Proposta Fácil


O que é a Aplicação Proposta Fácil?


Proposta Fácil visa automatizar o ato de realizar orçamento de mão de obra para pequenas e médias reformas, isso inclui pinturas e alvenaria, com uma interface simples a ferramenta traz opção de cadastro do cliente, calculo de mão de obra do serviço por m², incluindo ou não o material por parte do executor do serviço. Além de trazer para o profissional uma área de Dashboard para o controle dos orçamentos emitidos, tudo isso para ele ter um controle.

Problema:


Hoje para realizar um orçamento para uma Pequena ou Média reforma, seja ela em um residencia sala comercial, é preciso solicitar para um profissional como o empreiteiro autonomo vá até o endereço do cliente, para fazer as medições necessárias, identificar os tipos de serviços envolvidos, após ele precisa se debruçar nesses dados para chegar no valor a ser cobrado pela mão de obra, e se o cliente solicitar que na proposta comtemple o mateiral ele vai precisar calcular na mão as quantidades envilvidas.

Ideia:


A Ideia consiste em uma aplicação web, que permita a inserção dos dados do cliente, como nome, e-mail, telefones, endereço e observações, uma área para a inserção dos serviços aplicados com suas devidas quantidades e custo por unidade, uma área para a inserção dos materiais a serem utilizados no serviço, por fim gerar uma proposta/orçamento em formato PDF, para ser enviado ou entregue ao cliente.

Para o controle dos orçamentos emitidos, ganhos, perdidos, em orçamento, teremos uma área de dashboard para que o profissional consiga fazer uma análise da sua carteira de orçamentos e clientes.

Nicho:

Profissionais ou Clientes, dentro do nicho de profissionais temos, Pintores, Gesseiros, Carpinteiros, Assentador de Cerâmica, entre outros profissionais que prestam serviço de reformas, de forma autônoma Ou aqueles que possuem uma pequena empresa. Se tratando de clientes, temos todo o tipo de cliente que deseja planejar os custos de mão de obra e material para a sua reforma, antes mesmo de chamar um profissional.
